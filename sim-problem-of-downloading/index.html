Problem:
Symantec Installation Manager prompt for Domain\User name and password for “Reporsitory: www.solutionsam.com”

Reason:
It seems like related to something between your server and internet, in my case it’s a webfilter (Websense) that was blocking the download. How it allowed an initial install and blocked it later on is beyond me.
I see cases related to proxy server with user name and password. If you are also behind a proxy server, it worth a try to connect it directly to internet.
